二项式定理

二项式系数相关性质：

  • 对称性：Cnm=Cnn−mC_n^m = C_n^{n-m}Cnm​=Cnn−m​

  • 增减性与最大值：二项式系数前半部分逐渐增大，后半部分逐渐减小，中间取最大值。

    最大值max={Cnn2Cnn−12=Cnn+12\text{最大值} max=\begin{cases} C_n^{\frac{n}{2}} \\ C_n^{\frac{n-1}{2}}=C_n^{\frac{n+1}{2}}\end{cases} 最大值max={Cn2n​​Cn2n−1​​=Cn2n+1​​​

  • 二项式系数之和

    Cn0+Cn1+Cn2+Cn3+...+Cnn−1+Cnn=2nC_n^0+C_n^1+C_n^2+C_n^3+...+C_n^{n-1}+C_n^n=2^n Cn0​+Cn1​+Cn2​+Cn3​+...+Cnn−1​+Cnn​=2n

  • 二项式奇数项系数之和等于偶数项系数之和，即

    Cn1+Cn3+Cn5+...=Cn0+Cn2+Cn4+Cn6+...=2n−1C_n^1+C_n^3+C_n^5+...=C_n^0+C_n^2+C_n^4+C_n^6+...=2^{n-1} Cn1​+Cn3​+Cn5​+...=Cn0​+Cn2​+Cn4​+Cn6​+...=2n−1

CnmC_n^mCnm​计算

  • 方式一：公式计算
    计算都是在逆元或者阶乘基础上计算的

Cnm=n!m!∗(n−m)!C_n^m = \frac{n!}{m!*(n-m)!} Cnm​=m!∗(n−m)!n!​

ll C(ll n, ll m)
{
	if(n < m) return 0;
	return fact[n] * invfac[m] % mod * invfac[n - m] % mod;
}
  • 方式二：递推方式
    需要建表，所以如果计算范围比较大时需要的空间也大

Cnm=Cn−1m+Cn−1m−1C_n^m = C_{n-1}^{m} + C_{n-1}^{m-1} Cnm​=Cn−1m​+Cn−1m−1​

排列组合方法

1. 隔板法

引入： 10 个相同小球放进 3 个不同盒子，每个盒子至少放进一个， 求种类数。

在10 个小球中间插入隔板， 共有 10−1=910-1=910−1=9 个位置， 需要插入 3−1=23-1=23−1=2 个隔板将其分成 3 部分。 共 C92=36C_9^2 = 36C92​=36 种。

  • 一般隔板法：

将 nnn 个相同的元素分成 kkk 部分，每部分至少分得一个元素， 即在 n−1n-1n−1 个空隙中插入 k−1k-1k−1 个隔板，情况数为

Cn−1k−1C_{n-1}^{k-1} Cn−1k−1​

等价于 x1+x2+x3+...+xk=n,xi≥1x_1+ x_2+x_3+...+x_k=n, x_i \geq 1x1​+x2​+x3​+...+xk​=n,xi​≥1 的可行解个数。

  • 添元素隔板法：

将 nnn 个相同的元素分成 kkk 部分，每部分可以为空， 等价于我们先让元素个数添加上 kkk 个，然后在 n+kn+kn+k 个元素中分成 kkk 部分， 最后再在 kkk 部分的每一个部分都拿走一个元素，共拿走 kkk 个元素。 即在 n+k−1n+k-1n+k−1 个空隙中插入 k−1k-1k−1 个隔板，情况数为

Cn+k−1k−1C_{n+k-1}^{k-1} Cn+k−1k−1​

等价于 x1+x2+x3+...+xk=n,xi≥0x_1+ x_2+x_3+...+x_k=n, x_i \geq 0x1​+x2​+x3​+...+xk​=n,xi​≥0 的可行解个数。

此时该式可以转化为一般隔板法， 我们让 xi≥1x_i \geq 1xi​≥1 即可，操作是将每一个 xix_ixi​ 元素加上 111 ，相当于等式两边同时加了 kkk ，即 x1+x2+...+xk=n+k,xi≥1x_1+x_2+...+x_k=n+k,x_i \geq 1x1​+x2​+...+xk​=n+k,xi​≥1 ，即可转化为一般隔板法。
